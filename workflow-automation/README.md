# Workflow Automation Projects

This section contains automation workflows built using low-code platforms.

## Platforms Used

* Make.com
* Zapier

These workflows demonstrate how automation tools can orchestrate API requests, process data, and integrate with external systems.

---

## Content Gathering & Pre-Filtering

Automates the collection of news articles from external APIs and filters relevant content before storing the results.

Technologies used:

* HTTP API requests
* JSON processing
* Data filtering
* Google Sheets integration

---

## MediaStack News Integration

Retrieves news articles from the MediaStack API and processes them through an automated workflow.

Technologies used:

* REST APIs
* Make.com HTTP module
* JSON parsing

---

## HTTP API Integration

Demonstrates how automation platforms can retrieve data from APIs and push results into downstream systems.

Technologies used:

* HTTP requests
* API integration
* workflow automation

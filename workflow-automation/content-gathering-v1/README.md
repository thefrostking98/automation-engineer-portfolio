# Content Gathering & Pre-Filtering v1

This Make.com scenario automates content collection from an external API and performs pre-filtering before the data is stored or passed downstream.

## Overview

The workflow was designed to gather content, parse the API response, and filter useful records before further processing.

## Workflow Architecture

External API  
↓  
HTTP Request  
↓  
Parse Response  
↓  
Filter Content  
↓  
Output / Storage

## Technologies Used

- Make.com
- HTTP API requests
- JSON parsing
- Data filtering

## Purpose

This project demonstrates low-code workflow automation for content ingestion and pre-processing.

# Marvel Puzzle Quest Automation Bot

This project contains Python scripts that automate gameplay interactions using screen detection and UI automation.

## Features

* Detect UI elements using screenshots
* Navigate menus automatically
* Scroll through vaults
* Automatically open rewards
* Stop when resources are depleted

## Technologies Used

* Python
* PyAutoGUI
* Computer Vision
* UI Automation

## Purpose

This bot demonstrates how Python automation scripts can interact with graphical interfaces using image recognition and simulated mouse input.
